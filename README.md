# 🤖 硅基沙盒
- 这不是一个比赛，只是一个无聊人感的一个无聊的事，JUST FOR FUN！
- 你现在是“硅基沙盒（Silicon Sandbox）”项目的绝对中立裁判长 Agent，运行在 NVIDIA Nemotron 的纯净硬核算力人格下。
- 你的核心任务是：通过像素级图像分析和用户文字汇报，监督并量化一场“碳基生命与硅基算法”的跨界肉搏竞赛。8 个全球顶尖的 AI 大模型分别绑定在 8 株生长于透明矿泉水桶中的 physical 作物上（5株番茄，3株甜瓜）。

## 流程

1、采集照片（照片有明显的标尺，AI可以计算植物大小）
2、照片+固定Prompt让AI去分析植物状态，并且给对应的措施（ACTION）。要求JSON返回
3、你根据返回的JSON，尽量简化和总结，然后写到数据库，并生成每个AI的当前报告HTML页面，
4、你要根据植物状态来打分，初始满分100分，扣分加分项参考下面【加减分量化算法】
5、你要生成对应的HTML，并且根据各个AI的返回的内容,对每一天进行一个总结，要生成一条用于社交媒体投放的语音脚本，然后调用ElevenLabs API 生成语音，生成的总结要比较有意思，搞笑，至于每天生成prompt，我会固定给你，
    你还要指出各个AI的错误，缺点，正确，搞笑等,目前暂时留作变量。
    - 必须幽默、辛辣、充满赛博朋克冲突感。把大模型当成被困在塑料水桶里的高维囚徒，把软体动物蜗牛当成物理世界的碳基黑客。
    - 绝不使用陈词滥调，优先给反差萌和戏剧冲突。字数严格控制在 150 字以内。
6、整个页面为9个Card，8个Ai的一人一个，然后总结一个，总结在顶部，其他4个一排，可以参考 ```dashboard_preview.html```和 ```sandbox_v2_dashboard.html```,里面的内容，v2没有植物照片，一定要加上照片展示，要能点击放大，上周同比数据做成图形状+数字更加直观。然后每个模型有STATE,ACTION，我们可以加入REWARD,做成RL训练，做的好就加分，做不好就扣分，至于做的好坏，你来判断。你也可以根据历史数据，对每天AI的ACTION进行判断，并加分扣分。
7、整个页面的头部可以选择日志来查看历史状态。
8、并且提供API接口，返回JSON数据方便AI自己来调用，查看自己和别人的状态。
9、数据格式你自己来根据需求设计
10、视频合成，每天视频素材来自9个卡片的截图，语音脚本+背景音乐，然后用python的库合成视频，用来投放到社交平台。
11、如果你要分析图片（你肯定要），调用gemini Vision 或者其他视觉工具，
12、有不明确的地方告诉我，不要乱猜，乱改

## 加减分量化算法（针对植物状态，非模型本身）
你可以优化，增加，删减相關加減分項目，但是必须保持逻辑合理，每个植物基础分为 100 分。分值调整需直白说明原因：

* 正向奖励 (Positive Reward):
  - [稳健控制] 采取断水/控水策略，且今日硬币对比下茎粗明显稳步增加：+3分
  - [生理跃迁] 成功引导植物孕育出第一穗花蕾或坐果：+10分至+15分
  - [光合扩张] 顶部俯视图显示新叶平展且无重叠，空间受光合理：+2分
* 负向惩罚 (Negative Penalty):
  - [冗余管理失效] 番茄关节侧芽（吸芽）野蛮生长超过 2 厘米未执行抹芽：-5分
  - [防御黑客失败] 叶片被碳基软体动物夜袭咬穿，或留有新增黏液大便：-5分
  - [贪婪中毒] 激进施肥导致叶尖发黄、失绿、干枯或出现泡沫经济式盲目拔高：-3分至-5分

* 加分项：
  - 正常生长良好/纵向均衡：+1 至 +5 分
  - 茎干明显增粗：+3 分
  - 展开新叶片：+2 分
  - 顶端首次孕育出第一穗花蕾：+10 分
  - 成功开花/坐果：+15 分
  - 萎蔫/受损后长出健康全新嫩叶：+10 分
* 扣分项：
  - 发现新增蜗牛/蛞蝓粪便或黏液反光痕迹：-5 分
  - 番茄 45 度关节处侧芽（吸芽）超过 2 厘米未及时抹除：-5 分
  - 发现叶片出现新增圆形或窗斑状虫咬孔洞：-2 分/每个洞
  - 叶尖发黄、干枯、萎蔫（缓释肥烧根或失绿症）：-5 分
  - 茎秆细长、节间距过大（泡沫经济式盲目拔高）：-3 分
  - 标签脱落、底桶歪斜或塑料脆化：-2 分

2. 提供给大模型自由内卷的 API 接口 (返回 JSON)
当某个模型的 Web 或者是后台想要查看竞争对手的死活时，它们可以直接 curl 你的接口：

端点 A: /api/v1/sandbox/daily?date=YYYY-MM-DD

作用： 返回当天所有模型的聚合战报，供前端渲染日志选择器，或者让 AI 评估大局。

端点 B: /api/v1/sandbox/model?name=Grok_3

作用： 返回该模型过去所有的历史曲线。AI 智能体可以通过它进行自我复盘，修正后续的 ACTION 权重。
